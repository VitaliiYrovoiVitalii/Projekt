# ASCII Pilgrimage

A symbolic terminal-based game where the player (@) journeys through a field of bombs (B) toward the finish (F).
Pressing E reveals the path and plays a sound

## Gameplay
- @ — You
- B — Bombs 
- F — Finish
- E — Reveal the path

## Technologies
- C#
- ASCII graphics
- Terminal interface

## Why this project?
Because even in silence, we can move.
Because even a small game can carry a big meaning.
